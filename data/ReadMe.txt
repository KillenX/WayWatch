AdjMatrix.csv
_____________
Bottom triangular representation of adjacency matrix
- Cell (1,1) should contain number of nodes
- Cells with value '1' represent existing edges
- Empty cells represent non-existing edges

EdgeData.csv
_____________
Table with info about edges
- 1st row is header row, ignore in parsing
- 1st & 2nd col is edge represented by pair of connected nodes forming it
- 3rd & 4th col are distance and speed limit, respectfully
- Next cols are toll prices for each vehicle category
